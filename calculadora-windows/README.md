Trabalho da disciplina Web.


Descrição

Este projeto consiste em uma calculadora funcional desenvolvida em HTML5, CSS3 e JavaScript, seguindo o padrão visual e o comportamento básico da calculadora do Windows 10/11.
O objetivo foi implementar toda a lógica sem o uso de <input> ou <form>, trabalhando apenas com manipulação do DOM e eventos de clique e teclado.

Funcionalidades

A calculadora implementa:

Operações básicas

Adição (+)

Subtração (−)

Multiplicação (×)

Divisão (÷)

Entrada de dados

Clique nos botões

Teclado:

Números (0–9)

Operadores (+, -, *, /)

Vírgula (,) ou ponto (.) para decimais

Enter ou "=" para calcular

Backspace para apagar

ESC para limpar tudo

Controles

C – Limpa toda a operação

CE – Limpa apenas o número atual

⌫ – Apaga o último dígito

+/- – Inverte o sinal

% – Porcentagem com comportamento compatível com o Windows

Operações em cadeia (ex.: 5 + 3 + 2 =)

Exibição

Visor principal para o número digitado e resultado

Visor secundário mostrando a operação completa (ex.: 12 + 8 =)

Decimais exibidos com vírgula (,)

Tecnologias Utilizadas

HTML5

CSS3

JavaScript (DOM API nativa)

Estrutura do Projeto
index.html     -> Estrutura da calculadora
style.css      -> Estilização baseada no tema do Windows
script.js      -> Lógica da calculadora e eventos

Como Executar

Baixe ou clone o repositório

Abra o arquivo index.html diretamente no navegador

Utilize tanto o mouse quanto o teclado para operar a calculadora

Observações

O projeto foi desenvolvido sem uso de frameworks ou bibliotecas externas

Todo o comportamento foi implementado usando apenas JavaScript puro
